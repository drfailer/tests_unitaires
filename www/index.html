INSTRUCTIONS

Fonctionnement du site

Ce site web est destiner à tester vos réponce aux différents exercices donnés par vôtre professeur.
Utilisez le bouton en haut à gauche de la page pour faire défiler les différents menus qui seront afficher sur la partie droite de vôtre écran.

Lancement des tests

  1. Collez vôtre code réponce dans la zone de texte prévue à cet effet.
  2. Entez l'identifiant du jeux de tests à utiliser.
  3. Cliquez ensuite sur le bouton "Lancer les tests".
